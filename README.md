# Ebben Visuals — GitHub Pages Website

Cinematic videografie website met intakeformulier, automatische production breakdown en dashboard.

## Wat zit erin?

- Landing page
- Work/project types
- Client intake formulier
- Production breakdown generator
- Dashboard met projecten
- Responsive design
- Werkt volledig via GitHub Pages
- Geen installatie nodig
- Geen API key nodig

## Bestanden

Zet deze 3 bestanden in je GitHub repository:

```text
index.html
style.css
script.js
```

## GitHub installatie

1. Ga naar GitHub.
2. Maak een nieuwe repository.
3. Upload deze 3 bestanden:
   - `index.html`
   - `style.css`
   - `script.js`
4. Klik op `Commit changes`.

## GitHub Pages activeren

1. Ga naar je repository.
2. Klik op `Settings`.
3. Klik links op `Pages`.
4. Bij `Source`, kies `Deploy from branch`.
5. Kies branch: `main`.
6. Kies folder: `/root`.
7. Klik `Save`.

Je website komt online op:

```text
https://jouwgebruikersnaam.github.io/jouw-repositorynaam/
```

## Belangrijk

Deze versie gebruikt géén echte AI API. De breakdown wordt lokaal gegenereerd met JavaScript.

Waarom?
Een Claude/OpenAI API key mag je nooit rechtstreeks in frontend code zetten. Iedereen kan die sleutel dan zien en misbruiken.

Later kan je dit uitbreiden met:
- Tally formulier
- Notion database
- Make automatisatie
- echte AI breakdown via veilige backend
- custom domein
- portfolio video embeds
